# T10 — Chat en tiempo real

Chat completo con Express, MongoDB, JWT y Socket.IO.

## Ejecución

1. Copia `.env.example` a `.env` y completa `MONGODB_URI` y `JWT_SECRET`.
2. Ejecuta `npm install`.
3. Ejecuta `npm run dev`.
4. Abre `http://localhost:3000`.

Incluye registro, login, perfil, creación/listado de salas, historial persistente,
presencia online, usuarios por sala, entrada/salida, mensajes y estado
«escribiendo».
